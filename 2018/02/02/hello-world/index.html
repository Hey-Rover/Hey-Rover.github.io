Markdown 语法

Feb 2, 2018 | 语法

Markdown 的目标是实现「易读易写」。
不过最需要强调的便是它的可读性。一份使用 Markdown 格式撰写的文件应该可以直接以纯文字发佈，并且看起来不会像是由许多标籤或是格式指令所构成。
Markdown 的语法有个主要的目的：用来作为一种网络内容的写作用语言。
